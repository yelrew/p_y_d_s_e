PyDSE

PyDSE is a Python package for State Estimation of unbalanced Distribution Systems.

  • Source: https://github.com/yelrew/pydse

  • Bug reports: https://github.com/yelrew/pydse/issues

  • Documentation: https://yelrew.github.io/p_y_d_s_e_-_d_o_c

PyDSE operates in actual (non-per unit) network quantities. It provides:

  • An interface to read OpenDSS data files

  • Models to represent unbalanced 3-phase networks

  • Methods to perform load-flow analysis

  • Methods to perform state estimation in 3-phase networks, including Observability Analysis and Gross Errors check. Currently, the Branch-Current method for DSE is implemented.

Simple example

Setup a measurement set based on load-flow and run the Branch-Current Estimator on the IEEE-34 feeder in dss format:

>>> from pydse.dse import Dse
>>> dse = Dse(filename="IEEE34.dss")
>>> dse.addMeas('skm', location='all', accuracy=0.05)
>>> dse.addMeas('s-pseudo', location='load', ignore='prim', accuracy=0.5)
>>> dse.runDSE(report=True)

It is also possible to add real-time measurements with actual values:

>>> from pydse.dse import Dse
>>> dse = Dse(filename="data/IEEE13mod.dss")
>>> dse.addMeas_Realtime('skm', location='632-670', nodes='123', accuracy= 0.0,
value=[1.06725092e6+0.48933956e6j, 524.04365e3+125.60266e3j, 1.10045821e6+0.36163657e6j])
>>> dse.addMeas_Realtime('s-pseudo', location='611', nodes='3', accuracy= 0.0, value=163.5e3-15.54e3j)

Additional examples and data files can be found in the examples directory and the documentation .

Install

Install the latest version of PyDSE:

$ pip install -U git+ssh://git@github.com/yelrew/pydse

or:

$ git clone https://github.com/yelrew/pydse.git

$ cd pydse && pip install .
